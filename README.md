# PPG Signal Analysis System

## Опис проєкту

PPG Signal Analysis System — це програмна система для аналізу фотоплетизмографічних сигналів. Програма дозволяє завантажувати один або декілька сигналів, виконувати фільтрацію, знаходити піки серцевих скорочень, розраховувати BPM, HRV та індекс стресу.

## Основні можливості

- завантаження PPG-сигналів у форматах CSV/TXT;
- візуалізація сирого сигналу;
- цифрова фільтрація сигналу;
- детекція піків серцевих скорочень;
- розрахунок BPM;
- розрахунок HRV;
- оцінка рівня стресу;
- інтерпретація результатів;
- порівняння двох сигналів;
- збереження історії вимірювань;
- генерація PDF-звіту.
